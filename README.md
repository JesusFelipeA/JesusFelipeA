## 💼 ¿Qué hago?

- 🧩 **React Native (sin Expo)** – Apps móviles nativas con navegación, auth y escaneo QR/barra (CODE128).
- 🧠 **Laravel** – APIs RESTful, control de usuarios, reportes PDF, despliegue en hosting compartido.
- 🛠️ **Node.js + Express** – Backend sin tokens para apps móviles con persistencia de sesión.
- 🎨 **UI personalizada** – Estilo limpio y moderno inspirado en Bonafont.
- 🗃️ **Bases de datos** – MySQL/MariaDB, filtros dinámicos y consultas efectivas.
